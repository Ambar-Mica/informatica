Ejercicio: Mostrar / Ocultar texto

Este texto puede ocultarse o mostrarse.

JS (mostrar_ocultar.js) const boton = document.getElementById("btnToggle"); const parrafo = document.getElementById("parrafo"); boton.addEventListener("click", function() { if (parrafo.style.display === "none") { parrafo.style.display = "block"; } else { parrafo.style.display = "none"; }
